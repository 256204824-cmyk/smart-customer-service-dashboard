# 智能客服质量监控与培训 Agent 仪表盘

![深色数据仪表盘](https://img.shields.io/badge/Theme-Dark%20Dashboard-orange)
![License](https://img.shields.io/badge/License-MIT-green)

## 🎯 项目简介

一个专为智能客服质量监控与培训场景设计的可视化仪表盘，采用**深色数据风格** + **不对称网格布局** + **琥珀色点缀**。

## ✨ 核心功能

- **质量总览** - 综合质量分、CSAT、解决率实时监控
- **情感分析** - 正面/负面情绪占比环形图
- **质检评分分布** - 历史评分柱状图
- **实时会话监控** - 最新质检会话与缺陷热点
- **培训完成度追踪** - 培训进度仪表盘
- **AI 培训推荐引擎** - 基于缺陷的智能课程推荐
- **培训效果追踪** - 质检分趋势折线图

## 🛠️ 技术栈

- HTML5 + CSS3 (Grid 不对称布局)
- JavaScript (ECharts 数据可视化)
- Font Awesome 6 (图标库)
- Google Fonts (Inter 字体)

## 🚀 快速开始

1. 下载 `index.html` 文件
2. 直接用浏览器打开即可运行
3. 无需任何依赖或后端服务

## 📊 数据说明

仪表盘中的数据为**模拟演示数据**，实际使用时需对接真实 API 替换静态数值。

## 🎨 设计特点

- 深色半透明卡片 + 琥珀色高亮边缘
- 悬浮交互效果
- 响应式布局（桌面端优先，平板/手机自适应）
- 不对称网格视觉动感
